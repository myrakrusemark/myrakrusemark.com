Visual Auto Attendant

Project Overview

Role: UX Designer
Focus: Interactive voice response system visualization

The Challenge

Design an intuitive visual interface for configuring and managing complex interactive voice response (IVR) systems, making the traditionally technical task accessible to non-technical users.

Design Solution

Created a drag-and-drop visual builder that represents call flows as an interactive flowchart, allowing users to see and understand the entire customer phone experience at a glance.

Key Features

  • Visual flowchart representation of call paths
  • Drag-and-drop interface for building call flows
  • Real-time preview and testing capabilities
  • Pre-built templates for common scenarios
  • Integration with existing phone systems

Design Process

  • Shadowed users managing traditional text-based IVR systems
  • Mapped complex call flows to identify common patterns
  • Prototyped multiple visualization approaches
  • Conducted usability testing with business users
  • Refined based on feedback from technical and non-technical users

Impact

  • Reduced IVR configuration time by 60%
  • Enabled business users to make changes without IT support
  • Improved call routing accuracy through visual validation
  • Decreased customer complaints about phone navigation
